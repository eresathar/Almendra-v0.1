Introducción y origen de ALMENDRA

¿Qué es ALMENDRA?

ALMENDRA (Asistente de Lógica y Mejora Educativa Normalizada mediante Diagnóstico y Recomendación Adaptativa) es un software diseñado por y para el profesorado aragonés que simplifica y automatiza el diseño curricular bajo la normativa LOMLOE.

A través de una interfaz gráfica intuitiva y una base de datos local preconfigurada, el programa cruza saberes básicos, criterios de evaluación y descriptores operativos para estructurar el currículo de forma técnica; a partir de ahí, utiliza inteligencia artificial para generar automáticamente unidades didácticas objetivos y situaciones de Aprendizaje e informes de diagnóstico individualizados.
